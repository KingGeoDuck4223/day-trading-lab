# Day-Trading Lab — Journal

Full chronological record, one entry per trading day, newest at the top. This is the raw
evidence; memory.md is the distilled, rewritten-weekly summary drawn from it.

Around every 60 days, move entries older than that into log-archive.md to keep this file
readable, after making sure anything still relevant has been folded into memory.md.

Format per entry:

```
## YYYY-MM-DD — [strategy name] — [no trade / +X.XX% / -X.XX%]
- Strategy: [the exact rule used today, in plain language]
- Why this, why today: [continuing a streak / deliberate experiment / forced rethink
  after losses / brand new idea, and the reasoning behind it]
- What happened: [plain account of the day's scan, entries, exits]
- What went wrong (if anything): [be specific — vague answers aren't useful evidence]
- Next test: [what changes next time, and why]
```

On Fridays (or the week's last trading day), add:

```
### Weekly Summary
- Strategies tested this week and how each did
- What's looking promising vs. discredited
- What changed in memory.md and why
```
